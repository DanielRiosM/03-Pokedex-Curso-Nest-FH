# Run project

1. Clone repository
2. Run

```
$ npm run start:dev
```

3. Install NestJS CLI

```
$ npm i -g @nestjs/cli
```
4. Up Database

```
$ docker-compose up -d
```

# Stack used

- NestJS
- MongoDB
- Docker
- Docker Compose
